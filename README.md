# Duel Portfolio

Yu-Gi-Oh inspired single-page portfolio built with Next.js, Tailwind CSS, and Framer Motion.

**Live:** [zeeshanc.com](https://zeeshanc.com)

## Features

- Interactive duel disk navigation with holographic UI
- Cinematic contact section with 3D spinning card and power level bars
- Sound effects and ambient audio
- Fully responsive with reduced-motion support

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Tech Stack

- Next.js 15
- Tailwind CSS
- Framer Motion
- TypeScript
